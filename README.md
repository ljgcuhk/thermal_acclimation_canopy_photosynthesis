This repository contains all code scripts needed to reproduce the results and analysis presented in the following manuscript that is under review:
Jiangong Liu*, Youngryel Ryu*, Xiangzhong Luo, Benjamin Dechant, Benjamin D. Stocker, Trevor F. Keenan, Pierre Gentine, Xing Li, Bolun Li, Sandy P. Harrison, Iain Colin Prentice. Evidence for Widespread Thermal Acclimation of Canopy Photosynthesis. 2024. submitted.
